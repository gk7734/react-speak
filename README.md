# react-speak

> Made with create-react-library

## Install

```bash
npm install --save react-speak
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-speak'
import 'react-speak/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
